# 🧳 Urlaub-Packliste

## Dokumente
- [ ] Reisepässe / Personalausweise (alle)
- [ ] Versicherungskarten (Krankenversicherung)
- [ ] Buchungsbestätigungen
- [ ] Notfallkontakte

## Kinder (je Kind)
- [ ] Genug Kleidung für Urlaubsdauer + 2 Tage Reserve
- [ ] Pyjamas
- [ ] Badekleidung (2x)
- [ ] Sonnencreme
- [ ] Lieblings-Kuscheltier / Schnuller (Rosa)
- [ ] Bücher / Spiele für Reise
- [ ] Medikamente (falls regelmäßig)

## Erwachsene
- [ ] Kleidung
- [ ] Toilettenartikel
- [ ] Medikamente
- [ ] Lektüre

## Haus vor Abfahrt
- [ ] Fenster schließen
- [ ] Türen abschließen
- [ ] Strom/Wasser (ggf. absperren)
- [ ] Pflanzen gießen / Nachbarn beauftragen
- [ ] Post-Nachsendeauftrag oder Nachbarn bitten
- [ ] Hausmeister informieren
